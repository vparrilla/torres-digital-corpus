# Dataset description

## Overview

This document provides a technical description of the *Torres Digital Corpus*, a digital edition of the twenty-five solo counterpoint examples transmitted in Melchor de Torres’s *Arte ingeniosa de musica* (1566).

The authoritative, citable version of the dataset is archived on Zenodo (DOI: https://doi.org/10.5281/zenodo.17828789).


## Source
- Melchor de Torres, *Arte ingeniosa de musica* (Alcalá de Henares, 1566)
- Counterpoint examples transmitted in the original print

## Files and formats
The canonical dataset archived on Zenodo includes:

- PDF editions (modern score and analytical interval-labelled version)
- MusicXML transcriptions
- MIDI and MP3 renderings

This GitHub repository provides selected materials, primarily MusicXML files and documentation, for reuse and inspection.

## MusicXML files
Each MusicXML file corresponds to a single counterpoint example and follows a consistent naming scheme:
torres-001.xml
…
torres-025.xml

## Reuse
The dataset is intended for scholarly research, performance practice, pedagogy, and digital analysis of Renaissance improvised counterpoint.
