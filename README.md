# Torres Digital Corpus  
## The 25 Counterpoint Examples from Melchor de Torres (1566)

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.17828789.svg)](https://doi.org/10.5281/zenodo.17828789)

This GitHub repository provides supporting materials for the *Torres Digital Corpus*, a modern digital edition of the twenty-five solo counterpoint examples transmitted in Melchor de Torres’s *Arte ingeniosa de musica* (Alcalá de Henares, 1566). It is intended to facilitate inspection, reuse, and scholarly engagement with the corpus.

## Canonical dataset (Zenodo)

The authoritative, citable version of the *Torres Digital Corpus* is archived on Zenodo and should be cited in all scholarly contexts:

**DOI:** https://doi.org/10.5281/zenodo.17828789

Zenodo hosts the complete research dataset and ensures long-term preservation and stable citation.

## Relationship between Zenodo and GitHub

- **Zenodo** serves as the canonical archive of the dataset, including the complete set of PDF editions, MusicXML files, MIDI files, and audio recordings (MP3).
- **GitHub** hosts a curated selection of reusable materials derived from the dataset, together with documentation intended to support reuse, analysis, and further development.

This division reflects complementary roles: Zenodo as a preservation and citation platform, and GitHub as an environment for access, transparency, and reuse.

## Contents of this repository

This repository includes:

- Editorial documentation (Markdown)
- Navigational and overview tables (CSV)
- Individual MusicXML transcriptions of the twenty-five counterpoint examples
- Additional documentation related to reuse and extension of the corpus

Audio files and full-resolution PDFs are available exclusively through the Zenodo archive.
